Главная / Камуфляжи

Камуфляж — это внешняя специальная маскировочная окраска оружия, применяемая для уменьшения его заметности или придания ему некой уникальности, индивидуальности.

Камуфляж меняет только внешний вид оружия, его окрас, и не влияет на его характеристики. Убийства из оружия с надетым камуфляжем будут учитываться только в достижения используемого оружия.

После приобретения камуфляж можно установить на оружие, для которого он предназначается. Окно выбора камуфляжа находится в нижнем правом углу экрана при выборе оружия на складе. Для установки необходимо кликнуть по иконке нужного камуфляжа, а для возврата к оригинальной версии оружия — по специальному знаку отмены.

Оружейные камуфляжи не имеют срока годности и не требуют ремонта. Все имеющиеся у персонажа камуфляжи находятся на складе в разделе «Камуфляжи».
